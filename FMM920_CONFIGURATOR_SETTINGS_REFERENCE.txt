SECTION 11: TROUBLESHOOTING CONFIGURATOR
================================================================================

Cannot connect to device:
- Windows: Install COM port drivers (TeltonikaCOMDriver.exe)
- Check USB cable (must be data cable, not charging-only)
- Try Bluetooth: Pair as "FMM920_[last7digits]", PIN: 5555
- Verify device is powered (LEDs blinking)

Settings grayed out:
- Some settings require other settings enabled first
- Example: Static Navigation details grayed out when Disabled
- This is normal behavior

Configuration not saving:
- Verify device not in sleep mode
- Disconnect and reconnect
- Try Bluetooth instead of USB (or vice versa)
- Check Configurator version matches firmware

After saving, device not working as expected:
- Verify reboot completed (wait 3 minutes)
- Check Status tab for GPS fix and GPRS connection
- Review system log (Advanced â†’ System Info â†’ Logs)
- Reset to factory defaults and reconfigure if corrupted
